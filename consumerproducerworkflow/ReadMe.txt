Dependencies
================================================================================

    * You need the "java" and "javac" tools installed.

    * You need to have added the directory where "javac" resides to your
      PATH environmental variable.

    * Maven 


Instructions for compiling and running application
================================================================================

To compile the application open a console window and navigate to 
/../consumerproducerworkflow w and type:

   mvn compile

To run the application type:
 
   mvn exec:java -Dexec.mainClass="com.consumerproducer.workflow.ProducerConsumer" 

    
To delete all the class files generated by the javac compiler
 and install the artifact type: type

    mvn clean install
